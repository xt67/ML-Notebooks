# ML Notebooks

A collection of Machine Learning notebooks demonstrating regression techniques.

## Notebooks

### 1. Simple Linear Regression
- **File:** `1_Simple_Linear_Regression.ipynb`
- **Description:** Predicts appliance energy consumption based on temperature using Simple Linear Regression
- **Dataset:** `appliance_energy.csv`
- **Metrics:** Mean Squared Error (MSE), R-Squared

### 2. Logistic Regression
- **File:** `2_Logistic_regression_.ipynb`
- **Description:** Classifies sustainability of green technology based on various features using Logistic Regression
- **Dataset:** `green_tech_data.csv`
- **Features:** Carbon emissions, energy output, renewability index, cost efficiency
- **Metrics:** Accuracy, Confusion Matrix, Classification Report
